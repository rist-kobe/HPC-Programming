## 概要 / Overview
並列プログラミング入門（MPI編）の講義中で紹介されているサンプルプログラムのソースコードです。

This directory contains the source code for the sample programs introduced in the lecture *Parallel Programming Basics with MPI*.

---

## 環境要件 / Requirements
* Linux
* SLURM
* MPICH

---

## 実行方法 / How to Run
* Run the following command to build and submit all sample programs using SLURM:
```bash
bash exec_all.sh
```
